# 設定

此參考文件主要探討 Adobe Experience Platform Auditor 針對設定所執行的各項測試，為使用者提供詳細資訊。

設定測試會掃描您實作中的特定設定和值，找出潛在衝突。Platform Auditor 會根據其他規則和建議的最佳實務來評估標記。
