# 標記一致性

此參考文件主要探討 Adobe Experience Platform Auditor 為維持標記一致所執行的各項測試，為使用者提供詳細資訊。

Platform Auditor 的一致性測試會在所有掃描的頁面上尋找不一致之處。這些值或設定在網站的所有頁面上均應相同，以確保資料收集的正確性。
